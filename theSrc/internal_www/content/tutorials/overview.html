rHTML Labeled Scatter Example Tutorial

Intro

The rhtmlLabeledScatter is HTML widget that creates plots with a label placement algorithm.

This widget additionally allows the plotting of bubble plots and trend lines.

For example, the html source for this file: theSrc/content/tutorials/overview.html is used to generate this example R file: examples/tutorials/overview.r
